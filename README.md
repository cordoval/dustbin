# Dustbin README

Dustbin is a text and image pastebin.

Dustbin is built atop Symfony2.

# SYNOPSIS

1. Install following [this guide](http://symfony.com/doc/current/book/installation.html).
2. Request <http://example/p/new> in a web browser.
